# Web

Web interfaces for the BlueHub platform.

## Applications

- **admin/** - Admin dashboard for managing tenants, users, modules, and monitoring
- **client/** - Client portal built with Next.js 15, Shadcn UI, Tailwind CSS
- **shared/** - Shared UI components, utilities, and types used across admin and client

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Shadcn UI component library
- Tanstack Query for data fetching
