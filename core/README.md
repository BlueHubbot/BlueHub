# Core

Core business logic and shared services for the BlueHub platform.

## Subdirectories

- **auth/** - JWT authentication, password hashing, token management
- **users/** - User management service and schemas
- **billing/** - Billing and payment integration with Paymenter
- **rbac/** - Role-Based Access Control (permissions, roles)
- **audit/** - Audit logging for security and compliance
- **notifications/** - Notification delivery (email, Telegram, SMS)
- **license/** - License key management and validation
- **i18n/** - Internationalization engine (Persian, English support)
- **registry/** - Module discovery, registration, and feature flags
